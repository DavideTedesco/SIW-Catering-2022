## Use cases

### Admin use cases
- insert a new Chef
- insert a new Ingredient
- show all chefs, remove or edit a Chef
- show all Buffets
- show all Dishes
- show all Ingredients

### User use cases
- explore the selection by Buffets
- explore the selection by Chefs
