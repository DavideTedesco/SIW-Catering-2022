[See the app here!](https://siw-catering-2022.herokuapp.com/)

## Use cases

### Admin use cases
- insert a new Chef
- edit an Ingredient

### User use cases
- show Buffets by Chef
- show Dishes from a Buffet
- show Details of an Ingredient from a Dish
